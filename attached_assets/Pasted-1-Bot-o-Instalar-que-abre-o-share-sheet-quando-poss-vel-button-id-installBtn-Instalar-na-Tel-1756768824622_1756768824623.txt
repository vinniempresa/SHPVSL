1) Botão “Instalar” que abre o share sheet (quando possível)
<button id="installBtn">Instalar na Tela Inicial</button>
<script>
  const isiOS = /iphone|ipad|ipod/i.test(navigator.userAgent);
  const isStandalone = window.matchMedia('(display-mode: standalone)').matches || window.navigator.standalone === true;

  const openHowTo = () => {
    // Mostre um modal com instruções (fallback) se não houver Web Share
    alert('No iPhone/iPad: toque no ícone de compartilhar (quadrado com seta) e depois em "Adicionar à Tela de Início".');
  };

  document.getElementById('installBtn').addEventListener('click', async () => {
    if (isStandalone) return; // já instalado
    if (isiOS && navigator.share) {
      try {
        // Abre o share sheet do iOS; “Adicionar à Tela de Início” fica lá dentro (2 toques no total)
        await navigator.share({ title: document.title, url: location.href });
      } catch (e) { openHowTo(); }
    } else {
      openHowTo();
    }
  });
</script>

Service Worker básico (/sw.js)

self.addEventListener('install', e => {
  e.waitUntil(caches.open('v1').then(c => c.addAll(['/','/styles.css','/script.js'])));
});
self.addEventListener('fetch', e => {
  e.respondWith(caches.match(e.request).then(r => r || fetch(e.request)));
});


Registrar o SW

<script>
if ('serviceWorker' in navigator) {
  navigator.serviceWorker.register('/sw.js');
}
</script>

3) UX que ajuda (sem “burlar” regras)

Detectar se já está instalado (display-mode: standalone / navigator.standalone) e esconder o CTA.

Banner/tooltip contextual ensinando: “Toque em Compartilhar → Adicionar à Tela de Início”.

Ícones corretos (mín. 180×180 em /icons/apple-touch-icon-180.png) para o iOS mostrar um ícone bonito.

HTTPS obrigatório e service worker ativo para o PWA se comportar como app.
